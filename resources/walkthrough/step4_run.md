# Run: Execute Remote Workflows (Optional)

The [Run tab](command:nova-agent-vs-code-ext.showBuilderModeRun) lets you execute deployed workflows with custom payloads and monitor execution in real-time.

![Run Tab executing a workflow](step4_run.gif)

1. Select a deployed workflow from the list
2. Enter your JSON payload in the editor
3. Click "Run Workflow" to start execution
4. Watch logs stream in real-time
5. View your full Workflow Run in the AWS Console

**Tip:** You can run multiple workflows concurrently to compare their reliability and performance in remote environments.
