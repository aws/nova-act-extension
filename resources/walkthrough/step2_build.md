# Build: Develop and Test Locally

The [Build tab](command:nova-agent-vs-code-ext.showBuilderMode) is your local development environment. Build workflows cell-by-cell, debug in real-time, and test before deploying.

![Build Tab in action](step2_builder.gif)

1. Load Templates to choose a pre-built workflow for common use cases like form filling and data extraction
2. Run your script cell-by-cell to see results instantly
3. Watch your agent interact with websites in the live browser view
4. Review thinking & action logs to understand what your agent is doing

**Tip:** Use `@novaAct` in Copilot Chat to generate scripts from natural language.
